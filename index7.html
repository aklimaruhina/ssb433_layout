Text formatting

HTML contains several elements for defining text with a special meaning.Formatting elements were designed to display special types of text:

  • - Bold text b tag
  • - Important text strong tag
  • - Italic text i tag
  • - Emphasized text em tag
  • - Marked text mark tag
  • - Smaller text small tag
  • - Deleted text del tag
  • - Inserted text ins tag
  • - Subscript text sub tag
  • - Superscript text sup tag
